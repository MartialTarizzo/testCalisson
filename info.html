Présentation

Cette page propose de nombreuses grilles du jeu du Calisson, inventé par Olivier LONGUET.

Pour connaître la règle du jeu, s'informer sur les techniques de résolution et se confronter aux grilles proposées par l'auteur, aller sur le site du jeu.

Les grilles proposées ici (plusieurs centaines actuellement) ont toutes été conçues à l'aide d'un ensemble de fonctions programmées en Python, dont le code source est disponible à l'adresse suivante : https://github.com/MartialTarizzo/calisson

Normalement, toutes ces grilles ont une solution unique. Si jamais ce n'était pas le cas, c'est qu'il y a un erreur dans le code Python utilisé lors de la génération de ces grilles. Ne pas hésiter alors à m'envoyer un mail à l'adresse suivante :
martial.tarizzo (at) gmail.com

Mode d'emploi

  • sélectionner un groupe de grilles en cliquant sur un des boutons présents en haut de cette page. L'intitulé de chaque bouton est de la forme TAILLE.DIFFICULTÉ :
    le bouton "3.1" permet donc de jouer avec des grilles 'faciles' de taille 3 tandis que le bouton "6.2" permettra d'affronter des grilles nettement plus difficiles.
  • une liste de grilles est alors affichée à gauche de la fenêtre. Cliquer sur une des lignes pour afficher la zone de jeu permettant de tenter la résolution.
  • Il est possible de 'tricher' (à n'utiliser qu'en dernière extrémité, quand plus rien ne va...) :
    cliquer dans la zone de jeu avec la touche "Alt" du clavier enfoncée va afficher la solution !
    Re-sélectionner la grille dans la liste de gauche permet de recommencer la résolution ...

Pour afficher de nouveau ces informations, il suffit de cliquer sur le lien du titre de la page.

Le code de la zone de jeu a été modifié à partir de celui d'Arnaud DURAND utilisé sur la page d'origine.
